## 🚀 Sobre o Projeto
O **NpmProtect** foi desenvolvido para mitigar ataques de *Supply Chain* (Cadeia de Suprimentos). Ele analisa metadados de pacotes e comportamentos suspeitos antes que eles comprometam o seu ambiente de desenvolvimento.

### ✨ Funcionalidades
* 🔍 **Análise em Tempo Real:** Verificação de integridade de pacotes durante a instalação.
* 📦 **Multi-Linguagem:** Suporte inicial para ecossistemas Node.js (NPM) e Python (PyPI).
* 🛡️ **Threat Intelligence:** Integração com bases de dados de vulnerabilidades conhecidas.
* 🚀 **Lightweight:** Feito para rodar rápido no seu terminal Linux.
